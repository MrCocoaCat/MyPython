以源文件创建一个项目包
```
python setup.py sdist
```
将代码打包为tar.gz格式文件，方便管理
```
python setup.py install
```
进行代码安装


生成egg包
```
python setup.py bdist_egg
```
进行安装
```
easy_install
```
